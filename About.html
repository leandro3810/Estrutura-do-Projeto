📘 Sobre o Projeto

Este repositório foi organizado para servir como base de aplicações web com Flask no backend, TypeScript no frontend e uma estrutura simples para testes, automação e publicação estática.

🎯 Objetivos

  • Padronização: manter uma estrutura clara para backend, templates, arquivos estáticos e testes.
  • Automação: facilitar setup, lint e testes com scripts Bash versionados.
  • Qualidade: incluir validações com Ruff e Pytest desde o início do projeto.
  • Deploy: permitir uso tanto com Flask quanto com GitHub Pages para documentação estática.
