Sobre o Projeto: Estrutura-do-Projeto

Este repositório é uma base sólida para o desenvolvimento de aplicações web modernas, integrando um backend potente em Flask com um frontend dinâmico utilizando TypeScript.

Objetivos do Projeto

  • Padronização: Fornecer uma estrutura de pastas organizada e profissional.
  • Automação: Scripts Bash integrados para configuração rápida de ambiente virtual.
  • Qualidade: Configuração de linting com Ruff e testes automatizados com Pytest.
  • Pronto para Deploy: Estrutura compatível com GitHub Pages e servidores WSGI.

GitHub Pages

A interface de documentação e demonstração deste projeto é servida via GitHub Pages, garantindo acesso rápido e estático aos manuais de arquitetura.
