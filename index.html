MAPA DO CÓDIGO

Componentes e responsabilidades

Aplicação

  • python/__init__.py define a fábrica do app
  • python/routes.py concentra rotas web e API
  • python/config.py gerencia variáveis e segurança

Templates e páginas

  • templates/base.html estrutura visual base
  • templates/index.html página principal dinâmica
  • templates/about.html página institucional

Requisitos atendidos

  • Renderização de páginas principais com Flask
  • Assets estáticos compilados e servidos corretamente
  • Endpoint /health para monitoramento
  • Execução de lint e testes automatizados
Setup bash scripts/setup.sh
Build e execução npm run build && flask --app python/Run.py run
Qualidade bash scripts/lint.sh && bash scripts/test.sh && npm run build:check
