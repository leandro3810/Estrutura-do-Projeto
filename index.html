MAPA DO CÓDIGO

Onde está cada parte importante

Servidor

  • python/Run.py sobe o projeto Flask
  • python/routes.py organiza as rotas
  • templates/ entrega as páginas renderizadas

Interface

  • Style.css define a camada visual estática
  • Script.js controla o agente de IA do site
  • Static/style.css mantém a versão Flask alinhada

Operação

  • scripts/setup.sh prepara dependências
  • scripts/lint.sh roda Ruff
  • scripts/test.sh roda Pytest
Setup bash scripts/setup.sh
Build frontend npm run build
Executar Flask flask --app python/Run.py run --debug
