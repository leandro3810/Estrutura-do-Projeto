🚀 Estrutura do Projeto Flask + TypeScript

Este repositório contém uma estrutura organizada para o desenvolvimento de aplicações web utilizando Flask (Python) e TypeScript (JS).

🐍 Python 🌐 Flask 📘 TypeScript 🐚 Shell Scripts 🔍 Ruff

🛠️ Tecnologias Utilizadas

  • Backend: Python com Flask — servidor leve, roteamento e templates Jinja2.
  • Frontend: HTML5, CSS3 e TypeScript compilado para JavaScript.
  • Qualidade: Linting e formatação automática com Ruff; testes com Pytest.
  • Automação: Scripts Bash para configuração rápida do ambiente virtual.

📁 Estrutura de Pastas

python/

Lógica do servidor, rotas e configuração Flask.

templates/

Arquivos HTML com templates Jinja2.

Static/

Arquivos estáticos: CSS, JS compilado e imagens.

Tests/

Suíte de testes automatizados com Pytest.

scripts/

Scripts utilitários de automação Bash.

requirements.txt

Dependências Python do projeto.

⚡ Como Rodar

# 1. Configurar o ambiente virtual
bash configurar-ambiente.sh

# 2. Compilar os arquivos TypeScript
npm run build

# 3. Iniciar o servidor Flask
flask --app python/Run.py run --debug

# 4. Acessar no navegador
http://127.0.0.1:5000
