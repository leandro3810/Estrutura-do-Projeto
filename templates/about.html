SOBRE O PROJETO

Base organizada para site, código e automação

Este projeto reúne Flask, TypeScript e publicação estática em uma mesma estrutura, permitindo apresentar o código, navegar pelas seções do site e evoluir a aplicação com uma base mais clara para manutenção.

Objetivo

Criar uma fundação simples para sites institucionais, dashboards e páginas de documentação.

Agente IA

O frontend agora inclui um painel de comandos para orientar a navegação e destacar áreas do site.

Evolução

A estrutura está pronta para receber novas páginas, integrações e automações futuras.
