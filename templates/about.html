Sobre o Projeto

Este repositório é uma base sólida para desenvolvimento de aplicações web modernas, integrando um backend em Flask com um frontend dinâmico em TypeScript.

Objetivos

  • Padronização: Estrutura de pastas organizada e profissional.
  • Automação: Scripts Bash para configuração rápida do ambiente.
  • Qualidade: Linting com Ruff e testes automatizados com Pytest.
  • Frontend moderno: HTML5, CSS3 e TypeScript compilado.

Stack

Backend: Python 3 + Flask

Frontend: HTML5 · CSS3 · TypeScript

Ferramentas: Ruff · Pytest · Shell scripts
